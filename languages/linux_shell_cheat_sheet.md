# ⚙️ Linux / Shell Cheat Sheet

## 📁 File Management
```bash
cd /path/to/dir
ls -la
cp src dest
mv old new
rm -rf folder
```

## 🔐 Permissions
```bash
chmod +x script.sh
chown user:group file
```

## 📈 Monitoring
```bash
top
htop
df -h
du -sh *
```

## 🌐 Networking
```bash
ping google.com
curl http://example.com
netstat -tuln
```
