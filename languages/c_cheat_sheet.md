# 💻 C Programming Cheat Sheet

## 🔧 Structure
```c
#include <stdio.h>

int main() {
    printf("Hello, world!");
    return 0;
}
```

## 🔁 Loops
```c
for (int i = 0; i < 10; i++) {
    printf("%d\n", i);
}
```

## 📦 Functions
```c
int add(int a, int b) {
    return a + b;
}
```

## 🧠 Pointers
```c
int a = 5;
int *p = &a;
printf("%d", *p);
```
